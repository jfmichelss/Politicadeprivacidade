Política de Privacidade – SelfMirror

O aplicativo SelfMirror respeita a privacidade de seus usuários e está comprometido em proteger as informações coletadas durante o uso do aplicativo.

1. Coleta de informações

O SelfMirror pode coletar informações fornecidas voluntariamente pelo usuário, como textos, registros pessoais, imagens e preferências. Também podem ser coletados dados técnicos básicos necessários para o funcionamento do aplicativo, como modelo do dispositivo e versão do sistema operacional.

2. Uso das informações

As informações coletadas são utilizadas exclusivamente para:

  • Garantir o funcionamento correto do aplicativo
  • Melhorar a experiência do usuário
  • Personalizar recursos internos

O SelfMirror não vende nem compartilha dados pessoais com terceiros para fins comerciais.

3. Armazenamento e segurança

Os dados são armazenados de forma segura e são adotadas medidas técnicas para protegê-los contra acesso não autorizado, perda ou uso indevido.

4. Compartilhamento de dados

Os dados do usuário não são compartilhados com terceiros, exceto quando necessário para o funcionamento do aplicativo, como serviços de nuvem ou APIs utilizadas.

5. Crianças

O SelfMirror não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados pessoais desse público.

6. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomenda-se que o usuário revise esta página regularmente.
