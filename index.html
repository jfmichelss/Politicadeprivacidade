Política de Privacidade – SelfMirror

Última atualização: 2026

1. Sobre o Aplicativo

O SelfMirror é um aplicativo de uso pessoal voltado para reflexão individual utilizando inteligência artificial.

2. Coleta de Dados

O aplicativo:

  • Não exige cadastro
  • Não possui login
  • Não coleta CPF, CNPJ ou documentos
  • Não armazena dados pessoais em servidores próprios

3. Uso de Inteligência Artificial

O SelfMirror utiliza a API Gemini fornecida pelo Google para gerar respostas. Os textos inseridos pelo usuário podem ser enviados para processamento pela API, conforme os termos e políticas da própria Google.

Recomenda-se que o usuário não insira informações sensíveis ou pessoais durante o uso do aplicativo.

4. Armazenamento

O aplicativo não mantém banco de dados próprio. As informações permanecem apenas durante a sessão de uso.

5. Compartilhamento

O SelfMirror não vende, distribui ou compartilha dados de usuários com terceiros, exceto quando necessário para funcionamento da inteligência artificial.

6. Público-Alvo

Aplicativo destinado ao uso individual para fins de reflexão pessoal e entretenimento.
